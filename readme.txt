== Description ==

Lana Post Tags shows all post tags on edit sidebox.

Simple and useful plugin.

= Lana Codes =
[Lana Post Tags](http://lana.codes/lana-product/lana-post-tags/)

== Installation ==

= Requires =
* WordPress at least 4.0
* PHP at least 5.3
* JavaScript enabled in your browser

= Instalation steps =

1. Upload the plugin files to the `/wp-content/plugins/lana-post-tags` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
